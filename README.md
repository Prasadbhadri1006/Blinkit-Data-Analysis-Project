Blinkit Grocery Sales Data Analysis (Excel Project)
Overview

This project focuses on analyzing Blinkit’s grocery sales data using Microsoft Excel. It aims to uncover key sales insights, understand product performance, and evaluate how factors like outlet size, type, and location influence total sales. The project includes an interactive dashboard built using pivot tables, slicers, and charts for dynamic data exploration.

Objectives

Analyze Blinkit’s sales patterns across different outlets and product categories.

Identify top-performing product types and outlet formats.

Understand how fat content, visibility, and weight impact sales.

Create an interactive Excel dashboard for data-driven decision-making.

Dataset Information

Sheet Used: BlinkIT Grocery Data

Column	Description
Item Identifier	Unique ID for each grocery product
Item Type	Category of the item (e.g., Fruits, Canned, Snacks)
Item Fat Content	Fat classification (Low Fat / Regular)
Item Weight	Weight of the product in kilograms
Item Visibility	Shelf visibility ratio in stores
Outlet Identifier	Unique store ID
Outlet Type	Type of outlet (Supermarket Type1, Type2, etc.)
Outlet Size	Scale of outlet (Small / Medium / High)
Outlet Location Type	City classification (Tier 1, Tier 2, Tier 3)
Outlet Establishment Year	Year when the outlet was established
Total Sales	Sales value of each item
Rating	Average customer rating

Microsoft Excel

Data Cleaning and Formatting

Pivot Tables and Pivot Charts

Dashboard Design and Visualization

Key Insights

Supermarket Type1 outlets generated the highest total sales.

Medium-sized outlets consistently performed better than small or large ones.

Fruits & Vegetables and Snack Foods were top-selling product categories.

Regular-fat items slightly outperformed low-fat items in sales.

Outlets established between 2010–2015 achieved higher average sales.

Dashboard Features

Dynamic slicers for outlet size, location type, and product category.

KPIs for Total Sales, Average Rating, and Category Performance.

Visual representation of sales by year, outlet type, and category.

Fully interactive Excel dashboard for real-time insights.

Outcome

This project demonstrates how Excel can be used for data analysis and business intelligence. The dashboard enables Blinkit’s management to quickly interpret trends, identify growth areas, and make data-driven decisions effectively.
